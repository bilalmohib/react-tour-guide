# Changelog

All notable changes to this project will be documented in this file.

The format is based on [Keep a Changelog](https://keepachangelog.com/en/1.0.0/),
and this project adheres to [Semantic Versioning](https://semver.org/spec/v2.0.0.html).

## [1.0.0] - 2024-01-01

### Added
- Initial release of @tour-guide/react
- `Tour` component with full TypeScript support
- `useTour` hook for convenient state management
- Support for CSS selectors and React refs as targets
- Keyboard navigation (arrow keys, escape)
- Customizable styling and labels
- Auto-positioning tooltip with smart placement detection
- Spotlight/highlight effect for target elements
- Accessibility features (ARIA labels, keyboard support)
- Comprehensive callback system for tour events
- Zero legacy peer dependencies
- Full React 18+ support
- Tree-shakeable exports
- CSS modules support
- Examples and documentation

### Features
- Multiple placement options (top, bottom, left, right, center, auto)
- Progress indicator
- Skip and back navigation
- Overlay with customizable opacity
- Body scroll lock option
- Custom offsets per step
- Disable spotlight per step
- Disable scroll per step
- Smooth animations and transitions
